Prncipais formatações

Negrito / Destaque

Um termo pode ser destacado utilizando a tag B (não semântica).

Também pode-se utilizar a tag STRONG (semântica).

Itálico / Ênfase

Um termo pode apresentar ênfase utilizando a tag I (não semântica).

Também pode-se utilizar a tag EM(semântica).

Texto marcado

Pode-se marcar um texto utilizando a tag MARK.

Texto grande e pequeno

Texto grande (tag obsoleta) e texto pequeno.

Texto excluído

Pode-se marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Pode-se marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Para inserir coisas do tipo x2+3

Texto subscrito

Para inserir coisas do tipo H2O
